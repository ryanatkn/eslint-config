# eslint-config

Shared ESLint config (`@ryanatkn/eslint-config`).

## Committing

`git add` and `git commit` are denied by `.claude/settings.local.json` in
this repo — make the edits and stop, the user commits. As a public repo,
eslint-config is commit-denied per `~/dev/CLAUDE.md` §Committing. Never
bump versions or publish.
